# Convert V magnitude to illuminance in lux

Convert V magnitude to illuminance in lux

## Usage

``` r
vmag_to_ev_lux(mV, zero_point = 1)
```

## Arguments

- mV:

  Numeric vector of V magnitudes.

- zero_point:

  Optional scalar multiplier for catalog scaling.

## Value

Numeric vector of photometric illuminance in lux.
