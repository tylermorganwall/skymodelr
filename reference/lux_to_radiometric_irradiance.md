# Convert photometric illuminance to radiometric irradiance

Convert photometric illuminance to radiometric irradiance

## Usage

``` r
lux_to_radiometric_irradiance(E_v_lux, K_eff)
```

## Arguments

- E_v_lux:

  Illuminance in lux.

- K_eff:

  Effective luminous efficacy in lm/W.

## Value

Irradiance in W/m^2.
